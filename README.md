# Claude Code Workshop — Student Materials

歡迎來到 Claude Code Workshop！這個 repo 包含今天所有的練習素材。

## 快速開始

1. Clone 這個 repo 到桌面：
   ```bash
   cd ~/Desktop
   git clone https://github.com/museReed/claude-code-workshop-jr-student.git
   cd claude-code-workshop-jr-student
   ```

2. 依照講師指示完成設定，分兩階段：
   - **IDE 階段**：把 `agent-setup/to_IDE_AI_Agent_setup.md` 餵給 IDE 的 AI Agent（Antigravity）→ 裝好環境。
   - **CLI 階段**：回 terminal 啟動 `claude` 或 `codex`，把 `agent-setup/to_CLI_AI_Agent_setup.md` 餵給它 → 完成 GitHub 與面板。

| 資料夾 | 給誰 | 說明 |

### `agent-setup/`

| 檔案 | 給誰 / 何時 | 說明 |
|---|---|---|
| `to_IDE_AI_Agent_setup.md` | IDE Agent（第 1 階段） | 裝 Node + AI CLI（Claude Code / Codex）、提醒註冊 GitHub |
| `to_CLI_AI_Agent_setup.md` | CLI Agent（第 2 階段） | 啟動 claude/codex 後做 GitHub、面板 |
| `github-setup.md` | to_CLI 讀取的細部參考 | GitHub 設定、建立個人 repo 並 push 的細部指令 |
