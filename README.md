# Claude Code Workshop — Student Materials

歡迎來到 Claude Code Workshop！這個 repo 包含今天所有的練習素材。

## 快速開始

1. Clone 這個 repo 到桌面：
   ```bash
   cd ~/Desktop
   git clone https://github.com/museReed/claude-code-workshop-jr-student.git
   cd claude-code-workshop-jr-student
   ```

2. 依照講師指示，把 `setup.md` 餵給 Cursor AI 完成環境設定。

## 資料夾說明

| 檔案 / 資料夾 | 說明 |
|---|---|
| `exercises/` | 四個練習場景素材 |
| `setup.md` | 環境設定指令（給 Cursor AI 讀） |
| `github-setup.md` | GitHub 設定指令（給 Claude Code 讀） |
| `panel-setup.md` | 狀態列面板安裝指令 |
| `onboarding.md` | 新手操作小抄 |
| `faq.md` | 常見問題 |
