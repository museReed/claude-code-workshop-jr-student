# Claude Code Workshop — Student Materials

歡迎來到 Claude Code Workshop！這個 repo 包含今天所有的練習素材。

## 快速開始

1. Clone 這個 repo 到桌面：
   ```bash
   cd ~/Desktop
   git clone https://github.com/museReed/claude-code-workshop-jr-student.git
   cd claude-code-workshop-jr-student
   ```

2. 依照講師指示，把 `給AI_Agent安裝文件/setup.md` 餵給 Cursor AI 完成環境設定。
